What is RedPen?

RedPen is a tool to help writers or programmers who write technical documents or manuals that need to adhere to a writing standard. RedPen automates the verifications of input documents written in natural languages (NOT computer languages such as C++ or Java).


Note

This project is young, but we will update periodically as it is completed.
